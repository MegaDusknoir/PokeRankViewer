# PokemonHOME Rank Viewer

A Python GUI for viewing PokemonHOME battle usage rates.

一种基于 Python 的 PokemonHOME 使用率的 PC 端查询工具。

## 运行

创建程序图标
```
python ico2py.py
```
运行图形界面
```
python pokeHomeGui.py
```

## 封装

安装 pyinstaller
```
pip install pyinstaller
```
执行脚本
```
python release.py
```

## 使用

以暗色模式启动
```
--dark
```
